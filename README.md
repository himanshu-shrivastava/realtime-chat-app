### Setup .env file

````env
MONGODB_URI=
PORT=5001
JWT_SECRET=[can generate using `openssl rand -base64 32`]

CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=

NODE_ENV=development
````
*** 
### Build the app

```shell
npm run build
```

### Start the app

```shell
npm start
```
